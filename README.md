# python-college-project

A modern Breakout-style arcade game built with Python and Pygame. Control the paddle, destroy bricks, collect power-ups, progress through multiple levels, and achieve the highest score possible.

Features
Object-Oriented Architecture
Multiple Difficulty Levels (Easy, Normal, Hard)
Score Tracking System
Lives System
Multi-Level Gameplay
Random Power-Ups
Victory and Game Over Screens
Smooth Paddle and Ball Controls
Dynamic Ball Bounce Mechanics
No External Assets Required
Technologies Used
Python 3
Pygame
